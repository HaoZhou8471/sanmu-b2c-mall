# Sanmu Commerce B2C Mall Platform (hi-sm.com)

This project is an archive of the official B2C e-commerce website code developed and maintained for Sichuan Sanmu Technology Co., Ltd (Sanmu Micro Mall) during 2017-2018.

## Project Description

- **Development Background:** Responsible for the full-stack development and daily maintenance of the official online store of Chengdu Sanmu Commerce & Trade Co., Ltd.

- **Intellectual Property:** This platform obtained national computer software copyright registration in 2018 (Registration No.: 2018SR794918, see Figure 18).

- **Current Status:** Due to the passage of time and server changes, the database (SQL) files have been lost, and the `ecs_sessions` table in the system was corrupted 
- due to historical high concurrency access (see Figure 3). This project is mainly used for code logic archiving and front-end UI display.
